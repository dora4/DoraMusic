# DoraMusic

### 产品介绍

一款使用[https://github.com/dora4/dora] 开源库的Android手机音乐播放器。本代码仅限学习交流用途的分享传播，未经授权禁止用于其他用途。版权所有，侵权必究！

### 产品特性

- 基本的音乐播放控制
- 播放模式切换，包括单曲循环、列表循环、列表播放和随机播放
- 适用于音效调节的均衡器
- 音乐播放通知和通知栏控制
- 启动秒播
- 摇一摇切换歌曲
- 耳机监听与自动暂停
- 一键换肤
