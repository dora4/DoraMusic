# DoraMusic
一款使用dora框架[https://github.com/dora4/dora] 的Android手机音乐播放器，本代码仅限学习交流用途的分享传播，未经授权禁止用
于其他用途。版权所有，侵权必究。由于本项目主要作为框架的使用教程示范，部分API可能未能兼容到最新版本Android，敬请谅解！
