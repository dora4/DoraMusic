### 产品名称

朵拉音乐（英文名：Dora Music）

### 产品介绍

一个实用的Android手机音乐播放器。

### 产品特性

- 基本的音乐播放控制
- 播放模式切换，包括单曲循环、列表循环、顺序播放和随机播放
- 适用于音效调节的均衡器
- 音乐播放通知和通知栏控制
- 启动秒播
- 摇一摇切换歌曲
- 检测耳机与人性化的自动暂停
- 一键换肤（主题色）
- 单手操作

### 适用系统

Android7.0+

### 适用年龄

3+
 
### 支持语言

- 简体中文
- 英语

### 版本发布流程

1. 修改并提交代码，推送标签

2. CI/CD构建自动发布开发版到GitHub Release

3. 真机体验，如果不稳定则退回第一步，稳定则进入下一步

4. 从GitHub Release下载，然后上传稳定的用户版到蒲公英分发平台，如果和GitHub Release公示的sha256值不一致请不要使用
